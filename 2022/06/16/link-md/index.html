如何使用 npm link

功能

在本地开发 npm 模块时，我们可以使⽤ npm link 命令，将 npm 链接到对应的运⾏项⽬中去，⽅便的对模块进⾏调试和测试。

使用方法

首先创建两个项目，一个是npm-link-module ,我们要开发的npm模块；另一个是npm-link-example，是要运行npm模块的项目。

创建链接

进入npm-link-module，执行npm link

执行命令后，npm-link-module会根据 package.json 上的配置，路径是{prefix}/lib/node_modules/<package> ，这是官方文档上的描述，我们可以通过npm config get prefix

接着进入npm-link-example 项目中，执行npm link npm-link-module

npm-link-module 会被链接到npm-link-example/node_modules 下面，同样也是快捷方式。

到此我们创建成功。

核心命令

npm link : 创建软链接

npm unlink : 去掉软链接

注意事项

  • 当npm link npm-link-module报错的时候，比如下方的错误。可以使用相对路径npm link ./npm-link-module 就可以关联到

  • npm link npm-link-module 需要本地 build 一下

  • npm install 装的node-modules 会覆盖npm link
